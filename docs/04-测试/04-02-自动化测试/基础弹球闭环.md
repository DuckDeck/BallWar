# 基础弹球闭环自动化测试

> 版本归属：v0.2 | 生命周期状态：已审校

## 覆盖映射

| 测试资产 | 覆盖内容 |
|---|---|
| `t1_smoke_test.gd` | 首飞无重力、基础命中计分、障碍反弹系数 `1.36`、单球回合与棋盘基线 |
| `t1_rebound_speed_test.gd` | 障碍增强反弹后的速度上限；验证方向反转且速度不超过 `1400 px/s` |
| `t1_recovery_gate_test.gd` | 生命周期兜底仍须走完托底、侧边和顶部回程；抵达发射点前不得开始下一回合 |
| `t1_gravity_test.gd` | 首次碰撞前无重力；墙体碰撞后开启重力；托底入口与拖尾 |
| `t1_aim_guide_test.gd` | 圆形扫掠首个接触位置、一次短反射的方向与长度，及其与真实球反弹方向的一致性 |
| `t1_bottom_recovery_test.gd` | 左右托底回收、侧边上行、顶部回程与返回发射点 |
| `t1_timer_test.gd` | HUD 计时更新 |
| `t2_t3_architecture_test.gd` | 顺序多球、球间隔离、棋盘推进与危险线结束 |
| `t3_obstacle_variety_test.gd` | 六种障碍、中心旋转、匹配碰撞形状，以及按下一回合球数 `1×～2×` 生成数字 |
