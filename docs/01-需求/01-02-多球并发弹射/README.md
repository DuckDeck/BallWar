# 多球顺序弹射需求

> 所属功能域：多球弹射
> 版本归属：v0.3 | 生命周期状态：已审校

## 目标

一次发射操作可以控制多颗独立运行的弹球。首球立即从顶部缺口发射，余球在顶部两条斜线构成的槽内等待，并按同一瞄准角度依次发射。每球可独立反弹、命中障碍、触底回收，且拖尾颜色与球体一致；球与球之间不发生物理碰撞。

## 功能范围

- 一次有效发射创建一个可配置数量的球批次；奖励带来的永久球库存不设人工上限。
- 经典模式下，首球与余球均从顶部发射口按固定短间隔逐颗发射；待发球仅显示为顶部槽内预览，不创建物理实体。挑战模式也复用同一顶部凹槽预览，但凹槽始终开放，场内存在其他活动球时仍可在顺序队列清空后发起下一批。
- 同批球使用首球的原始瞄准方向，不使用扇形偏转；首飞阶段保持该直线方向，首次命中墙或障碍后才启用重力。
- 每颗活动球有独立的位置、速度、重力开关、碰撞结果、拖尾和回收状态。
- 每球使用自身显示色，拖尾使用相同 RGB，仅透明度与宽度渐变。
- 所有活动球可碰撞场地边界和障碍；球不检测或影响其他球。
- 仅当待发队列为空且所有活动球均回收后，才进行一次回合结算并恢复发射。

## 业务规则

- **L1-MB-01**：一次发射的每颗球具有独立运行状态；任何球的命中、回收或销毁不得停止其他球。
- **L1-MB-02**：待发球必须显示为顶部队列，且不创建碰撞体；发射后，余球按发射节拍依次补到发射口并递减。
- **L1-MB-03**：同批每颗球必须沿首球瞄准方向发射；不得出现扇形偏转。
- **L1-MB-04**：每颗球在首次墙体或障碍碰撞前不受重力影响；首次碰撞完成后，该球在后续运动中持续受重力影响。
- **L1-MB-05**：活动球仅检测场地边界和可受击障碍，不检测任何其他球。
- **L1-MB-06**：球体主色和拖尾主色来自同一个球视觉定义；同批允许出现不同颜色。
- **L1-MB-07**：经典模式在待发队列和活动球集合均清空前不得再次发射；先回收的球不影响其他活动球。挑战模式当前顺序队列清空后，只要发射口有已回收待发球即可再次发射。球只有实际抵达顶部发射点后才能从活动集合注销，超时或越界兜底也必须走同一回收路径。
- **L1-MB-08**：队列与活动集合清空后，系统只结算一次本回合并恢复 `READY`。
- **L1-MB-09**：经典模式在本批存在活动球时关闭发射口。每颗球完成顶部回收后进入两条斜线之前的暂存堆；仅在最后一颗球回收且安全结算完成后打开口，由首球先落入发射口、余球逐颗自然补位。堆叠只用于视觉，不产生球间碰撞。
- **L1-MB-10**：挑战模式不使用经典模式的闭口暂存规则；任一回收球抵达顶部发射原点后立即进入始终开放的待发凹槽。凹槽中的球同样受重力、斜坡与轻微碰撞影响；最低球自然占据发射口，其他球被上方凹槽支撑。当前顺序队列清空后，发射口有球即恢复发射资格。
- **L1-MB-11**：两种模式的顶部预览球均须表现为被抛入凹槽后的自然运动：左侧回收球上冲后获得向右的内向速度，右侧回收球上冲后获得向左的内向速度；二者均受重力回落至斜坡、向中心滑动，并与其他预览球发生低反弹推挤。碰撞动能持续衰减，最终固定为稳定堆叠。经典模式仅在最后一颗球回收且安全结算后打开凹槽；挑战模式始终保持凹槽开放。两种模式均只能让最低球实际滑入发射口；槽位已有球时，其他球仍由发射口上方的虚线凹槽底部支撑，不得继续下落；每次发射后，下一球也须实际补位至该口才可发射。
- **L1-MB-12**：球定义可为普通球或重球。重球半径为普通球的 `130%`，具有更高的可配置重力倍率；每次命中障碍时独立以可配置概率（默认 `15%`）造成 `2` 点伤害，否则造成基础 `1` 点伤害。球体、碰撞体、拖尾、瞄准扫掠与待发预览必须使用同一半径倍率。
- **L1-MB-13**：奖励块占用一个原本会生成普通障碍的逻辑格。任意奖励块被球命中后立即移除，不得得分、不得反弹，命中球保持原有速度和轨迹。`+1` 奖励立即在奖励格生成一颗零初速度、受重力的普通球，并将其永久加入本局后续批次库存；放大奖励被普通球命中时，命中球原地升级为重球并保持轨迹；被重球命中时，立即额外生成一颗零初速度、受重力的重球，并将其永久加入本局库存。新增球归入命中球所在批次，不能使该批提前结算。
- **L1-MB-14**：永久球库存没有数值上限；每次有效的 `+1` 或重球复制奖励都必须增加一颗球。性能调优不得通过静默丢弃奖励球实现。

## 非范围

本需求不定义爆炸、穿透、分裂，也不定义奖励格生成/下压规则、暂停 UI、Android TV 输入或对象池实现。

## 验收方向

- 经典模式起始球数设为 10 时，开口动画结束后首球位于发射口、余球在顶部队列可见；发射后按节拍逐颗补位并离开。
- 后续球以固定短间隔依次出现，均沿首球瞄准方向发射，球间无碰撞。
- 首次命中前球轨迹与瞄准方向一致；第一次墙体或障碍碰撞后，重力使其轨迹下坠。
- 经典模式的回收球在顶部斜线前可见堆叠，最后一球回收并安全结算后才打开下一次发射；挑战模式凹槽始终开放，任一回收球进入同样的自然堆叠预览，顺序队列清空后可立即恢复其发射资格。
- 重球在场内和顶部预览均明显大于普通球；将其双倍伤害概率分别设为 `0%` 与 `100%` 时，每次命中应稳定为 `1` 与 `2` 点伤害。
- 命中 `+1` 奖励后，原球不反弹，新普通球自然下落；下一批待发库存永久增加 1。普通球命中放大奖励后继续沿原轨迹成为重球；重球命中时额外自然下落一颗重球，且下一批库存增加 1。
