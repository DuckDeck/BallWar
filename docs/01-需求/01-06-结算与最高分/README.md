# 结算与最高分需求

> 所属功能域：局内流程、结算与本地记录  
> 版本归属：V0.4 | 生命周期状态：已审核

## 目标

对局到达危险线而结束时，玩家应立即看到本局成绩以及本模式的历史最高分、今日最高分，并可重新开始相同模式或返回启动后的模式选择页。

## 核心规则

- **L1-GO-01**：经典模式和挑战模式各自独立记录历史最高分与今日最高分；一方的成绩不得影响另一方。
- **L1-GO-02**：本局结束时，以本局最终分数更新对应模式的历史最高分；仅当本机日历日期相同，今日最高分才与此前记录比较。日期变化后，首次结算从零重新累计今日最高分，历史最高分始终保留。
- **L1-GO-03**：结算出现后，场内画面、物理、计时和波次推进必须冻结，且不再接受发射或暂停。
- **L1-GO-04**：结算面板展示“本局结束”、本局分数、历史最高和今日最佳；采用参考图的深色遮罩、青绿色标题条、深蓝成绩卡、装饰星形和底部三枚圆形操作入口。
- **L1-GO-05**：返回操作清除本局运行状态，进入应用启动后的模式选择页；经典模式与挑战模式按钮均可再次选择。
- **L1-GO-06**：重新开始操作使用本局模式创建新对局，清除当前分数、棋盘、活动球、计时和波次状态。
- **L1-GO-07**：分享/炫耀入口只作为视觉占位，当前禁用且不得触发分享、跳转或状态改变。

## 非范围

- 本切片不实现云端排行榜、账户同步、玩家百分比、奖励货币或实际系统分享。
- 本地最高分不替代后续“保存并退出”的对局续玩存档。
