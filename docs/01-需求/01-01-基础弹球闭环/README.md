# 基础弹球闭环需求

> 所属功能域：基础弹球闭环
> 版本归属：v0.2 | 生命周期状态：已审校

## 目标

玩家从顶部缺口向下瞄准发射弹球，利用墙体和障碍反弹完成命中，球最终经托底回收并回到顶部。界面始终显示分数与计时。

## 业务规则

- **L1-BB-01**：发射原点位于 `(540, 450)`，即出口下方一个球直径；向左下或右下发射不得首帧撞到出口挡板。
- **L1-BB-02**：引导使用与真实球相同半径的圆形扫掠，主线必须停在球心的首次安全接触位置，而非球心射线碰到墙面的中心点。
- **L1-BB-03**：主线命中后绘制一段较短反射虚线，反射法线必须来自同一次圆形扫掠的接触信息；仅预测一次反射。
- **L1-BB-04**：球在首次世界碰撞前严格沿瞄准方向飞行，不受重力；首次撞墙或障碍后，从下一物理帧起持续受重力影响。
- **L1-BB-05**：障碍反弹速度保留系数为 `1.36`，即此前 `0.68` 的两倍；墙体仍保持原速度。每次反弹后球速上限为 `1400 px/s`，避免连续碰撞时倍率相乘而越弹越快。
- **L1-BB-06**：球到达托底曲线后不再物理反弹，改为滚向侧边、沿外侧上行并回到顶部发射原点。任何超时或越界兜底也只能进入同一回收路径；只有球实际抵达发射原点后，才允许批次结算和下一回合开始。

## 非范围

不包含多次反射预测、轨迹仿真中的重力曲线、特殊球效果、暂停与结算 UI。
