# 基础弹球闭环需求

> 所属功能域：基础弹球闭环  
> 项目形态：离线 2D 客户端  
> 版本归属：V0.1-MVP | 生命周期状态：已审核

## 产品目标与用户价值

用户能够在竖屏场地中从顶部发射一颗球，观察其向下运动、反弹并击中数字障碍物。一次完整操作让用户获得“瞄准—命中—消除—得分—下一次瞄准”的即时反馈，是后续多球和关卡玩法的最小基础。

## 功能范围

- 发射：用户可在顶部中央缺口下方的任意屏幕位置点击，或从该区域向下拖拽并松开；球始终从该缺口沿目标方向向下发射。
- 反弹：球碰到场地的左右或顶部边界后继续在场内运动。
- 重力：活动球持续受向下重力影响；向上反弹后必须逐渐减速、到达最高点并回落。
- 障碍：场内存在一枚初始生命值为 3 的数字障碍物；每次命中使生命值减少 1。
- 得分：障碍物生命归零后被清除，Score 增加 10。
- 回合：球触到底部浅托底曲线后，按落点向对应侧滚出场地、在侧墙外上行并沿顶部斜线回到发射缺口；抵达缺口后本回合结束。停留超过最大时长或越过异常兜底线时也必须结束本回合；如障碍物已清除，则在下一回合开始前补充一枚初始障碍物。

## 业务流程

1. 初始状态显示发射位、一枚数字障碍物与 Score 0，用户可瞄准。
2. 用户在顶部中央缺口下方点击或完成向下拖拽，球从缺口进入场地并开始反弹。
3. 球命中障碍物时，障碍数字减 1；数字为 0 时障碍消失且 Score 增加 10。
4. 球触及底部浅托底曲线后，停止受重力和反弹影响；按落点左右分流滚出侧墙、在墙外上行、沿同侧顶部斜线回到中央缺口。只有活动球抵达缺口时，当前回合才结束。
5. 若球达到最大存活时长或越过异常兜底线，则立即按异常回收结束当前回合。
6. 系统恢复至下一回合的可瞄准状态；若已消除障碍物，则补充一枚新的初始障碍物。

## 业务规则

- **L1-BW-01**：一次可用回合只允许发射一颗球；球在场时不可再次发射。
- **L1-BW-02**：顶部中央缺口上方的按下或松开操作不产生发射；缺口下方的单击不受最小拖拽距离限制。
- **L1-BW-03**：每次有效球体命中仅使当前障碍物生命减少 1。
- **L1-BW-04**：只有障碍物生命降为 0 才增加 10 分；普通命中不加分。
- **L1-BW-05**：球回收或超时后必须恢复可发射状态，不能停留在中间状态。
- **L1-BW-06**：球体的竖直速度每个物理帧都叠加向下重力；碰撞反弹不重置速度大小。
- **L1-BW-07**：球命中六边形等障碍物时，仅保留 `obstacle_bounce_restitution` 比例的反弹速度；场地边界不受此衰减影响。
- **L1-BW-08**：活动球必须显示沿实际运动路径向后收尖、渐隐的拖尾；拖尾仅为视觉反馈，不改变球的碰撞、速度或回合判定。
- **L1-BW-09**：处于可发射状态时，顶部中央缺口必须显示一颗待发球；发射后待发球隐藏，当前球回收并恢复可发射状态后再显示。

## 文字线框

```text
顶部：两条斜边边框在中央留出缺口并以短竖线收口；球从该缺口中心向下发射。Score 位于左侧分隔虚线正上方；缺口下方整屏为点击/拖拽发射区
中部：深色场地，放置一枚带数字的青色几何障碍物
底部：弧形分隔与回收区
```

## 异常边界

- 当球的运动超过最大存活时长时，按回收处理，避免单回合无法结束。
- 本阶段没有失败、暂停、奖励、多球、关卡下压或存档规则；这些能力不应由本需求推断出来。

## 2026-07-19 托底回收补充

- **L1-BW-10**：球首次接触底部浅托底曲线时，立即停止重力和反弹；落点在场地中心左侧时向左滚动，右侧时向右滚动。
- **L1-BW-11**：球滚出对应侧墙后，须在墙外上行，经该侧顶部斜线实际移动到中央缺口发球点，才结束本回合并恢复待发球球。
- 托底回收动画期间不产生二次反弹，也不允许再次发射。
