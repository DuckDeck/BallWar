# 基础弹球闭环需求

> 所属功能域：基础弹球闭环
> 版本归属：v0.2 | 生命周期状态：已审校

## 目标

玩家从顶部缺口向下瞄准发射弹球，利用墙体和障碍反弹完成命中，球最终经托底回收并回到顶部。界面始终显示分数与计时。

## 业务规则

- **L1-BB-01**：发射原点位于 `(540, 450)`，即出口下方一个球直径；向左下或右下发射不得首帧撞到出口挡板。
- **L1-BB-02**：点击或拖拽时，引导主线从发射原点延伸至第一个 WORLD 墙体或障碍命中点；若没有命中，则延伸至配置最大长度。
- **L1-BB-03**：主线命中后绘制一段较短反射虚线，方向由命中法线计算；仅预测一次反射，不计算第二次命中或反射。
- **L1-BB-04**：球在首次世界碰撞前严格沿瞄准方向飞行，不受重力；首次撞墙或障碍后，从下一物理帧起持续受重力影响。
- **L1-BB-05**：球命中边界或障碍时反弹，障碍生命归零后只计分一次。
- **L1-BB-06**：球到达托底曲线后不再物理反弹，改为滚向侧边、沿外侧上行并回到顶部发射原点。

## 非范围

不包含多次反射预测、轨迹仿真中的重力曲线、特殊球效果、暂停与结算 UI。
