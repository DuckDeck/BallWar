# 游戏音频需求

> 所属功能域：局内反馈与音频  
> 版本归属：V0.4 | 生命周期状态：已审核

## 目标

为两种模式提供可辨识但不干扰多球玩法的背景音乐和关键事件音效。音频资源随 Godot/Android 导出包发布，不依赖网络或外部文件路径。

## 核心规则

- **L1-AU-01**：经典模式播放经典 BGM，挑战模式播放挑战 BGM；BGM 必须循环。
- **L1-AU-02**：开始一批发射、球回收、命中数字障碍、收集奖励格时分别播放对应音效。
- **L1-AU-03**：暂停和游戏结束时暂停 BGM；继续游戏时恢复，重开、返回模式选择和保存退出时停止。
- **L1-AU-04**：多球并发时命中与回收音效应进行短间隔限流，避免同帧重复叠加；不同类别音效仍可并行播放。
- **L1-AU-05**：音乐和音效音量必须在全局音频管理器中独立可调。

## 资源映射

| 项目资源 | 游戏用途 |
|---|---|
| `assets/audio/classic_bgm.wav` | 经典模式循环 BGM |

| `assets/audio/launch.wav` | 整批发射 |
| `assets/audio/recover.wav` | 球回收至顶部 |
| `assets/audio/block_hit.wav` | 数字方块受击 |
| `assets/audio/reward.wav` | 奖励格收集 |
