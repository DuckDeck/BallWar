# 暂停与局内状态需求

> 所属功能域：局内流程、暂停与 HUD  
> 版本归属：V0.4 | 生命周期状态：已审核

## 目标

玩家在一局经典或挑战模式中可随时暂停、继续或重新开始。暂停是本局内的临时冻结，不改变球、棋盘、分数、剩余时间或待发球库存；重新开始则以当前模式创建全新一局。

## 核心规则

- **L1-PS-01**：仅在已经开始且尚未结束的对局中允许暂停，涵盖 `READY`、`BALLS_ACTIVE` 与 `RESOLVING`；模式选择与 `GAME_OVER` 不显示或接受暂停。
- **L1-PS-02**：暂停后必须冻结场内物理、球的回收/发射队列、棋盘推进、单局正计时，以及挑战模式的下一波倒计时；不得在暂停期间生成或移动任何游戏对象。
- **L1-PS-03**：继续游戏后恢复暂停前的原状态和所有时间线，不重置计时、分数、波次、球位置或挑战倒计时。
- **L1-PS-04**：暂停菜单在冻结状态下仍可响应“继续游戏”和关闭操作；关闭与继续等价。
- **L1-PS-05**：暂停菜单的重新开始必须保留当前模式，但清除活动球、顺序发射队列、棋盘节点、波次 Tween 与挑战时钟；分数、回合、正计时、批次编号和本局球库存均回到新局初值。
- **L1-PS-06**：暂停菜单的“保存并退出”保存当前模式的可恢复续局状态并回到模式选择；具体恢复边界见《续局存档与 HUD 需求》。

## 非范围

- 系统级原生返回键的页面级交互不在本切片定义；后台/退出时的本地续局持久化由《续局存档与 HUD 需求》定义。
- 不定义暂停音效、入场动画或最终主题资源。
