# T1 托底回收施工蓝图

> 状态：已完成
> 依据：`T1-托底回收轻量设计方案.md`（用户已确认完整顶部回归动画）

## 1. 施工边界

- 把底部曲线移到侧墙终点下方，改为由同一份 `GameConfig` 参数驱动的浅托底曲线。
- 活动球接触托底曲线后，停止重力与反弹，按落点向左或向右滚出场地。
- 球滚出侧墙后，在墙外上行；到达顶部后沿对应斜顶移动至中央缺口，实际抵达发球点才回收。
- 不改发射方向、障碍物规则、计分规则、多球、广告区或 Android TV 输入。

## 4. 对抗评审

- 风险：视觉曲线与命中曲线漂移。缓解：两者均调用 `GameConfig.get_bottom_trough_y()`。
- 风险：回收动画再次与侧墙碰撞。缓解：滚动、上行、顶部回归均显式更新坐标，不运行物理碰撞。
- 风险：球在顶部被过早删除。缓解：只有最后一个路径点 `launcher_position` 抵达后才发出 `recovered` 信号。
- 风险：已有重力调参被覆盖。缓解：不改 `ball_gravity = 4000.0`。
