# T1 方向纠正记录

> 日期：2026-07-18  
> 状态：已完成  
> 决策来源：用户现场运行反馈

## 已拍板规则

- 发射位固定在场地顶部中部。
- 发射圆圈下方的整屏区域均可点击或拖拽；无需点中圆圈。球始终从顶部发射位沿松开位置方向向下发射。
- 底部弧形区域仅承担球体回收，不再是发射或瞄准入口。

## 变更范围

- `resources/game_config.tres` 与 `scripts/game_config.gd`：发射位置改为 `Vector2(540, 460)`。
- `scenes/main.tscn`：`Launcher` 移至顶部发射位置。
- `scripts/launcher.gd`：合法方向改为 `y > 0`，并将激活范围扩展到发射圆圈下方整屏；瞄准线与目标方向同向。
- `tests/t1_smoke_test.gd` 与 L1/L2/L5/L7、项目概览、任务总控：同步新方向和整屏发射入口。

## 验证

使用 Godot `4.7.stable.mono.official.5b4e0cb0f` 无头运行 `res://tests/t1_smoke_test.gd`：通过侧墙反弹、三次伤害、障碍清除得分和连续三回合回归。

## 重力补充（2026-07-18）

- `GameConfig.ball_gravity` 作为统一可调参数，当前值为 `1400.0`。
- `Ball` 在每个物理帧累积向下重力，反弹保留当前速度大小，不再归一化为固定球速。
- `tests/t1_gravity_test.gd` 验证上弹减速、到顶后回落；完整 `t1_smoke_test.gd` 同时通过。
