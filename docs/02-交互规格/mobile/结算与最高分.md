# 结算与最高分交互规格

> 所属功能域：局内流程、结算与本地记录  
> 版本归属：V0.4 | 生命周期状态：已审核

## 结算视觉

- 游戏背景保持终局画面并覆盖深色半透明遮罩。
- 居中成绩卡使用深蓝圆角底板与青绿色圆角标题条；标题为“本局结束”，卡片两侧添加青绿色星形装饰，整体参考提供的结算图。
- 标题下居中显示大号白色本局分数，其下为当前模式名称；中部两行展示“历史最高 / 今日最佳”及对应数字。
- 底部横向排列三个实心圆形入口：红色返回、黄色炫耀、绿色重新开始。图标必须在各自圆心垂直居中；中文说明作为独立文本行，位于圆形正下方并以圆心对齐，不能作为圆形按钮的溢出子内容。图标优先使用 `BallWarIconFont.ttf` 的返回 `E641`、分享 `E663`、刷新 `E689`。

## 操作状态

| 操作 | 可用性 | 结果 |
|---|---|---|
| 返回 | 可用 | 关闭结算，回到模式选择页 |
| 炫耀 | 禁用 | 保留黄色圆形视觉，不响应点击 |
| 重新开始 | 可用 | 关闭结算，以当前模式开启新局 |

- 结算遮罩吞掉卡片外点击，不能透过面板继续拖拽瞄准或操作 HUD。
- 结算 UI 使用不受 `SceneTree.paused` 影响的处理模式，保证返回和重开在冻结世界上方可点击。
