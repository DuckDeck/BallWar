# 续局存档与 HUD 交互规格

> 所属功能域：局内流程、HUD 与本地续局  
> 版本归属：V0.4 | 生命周期状态：已审核

## HUD 结构与适配

- HUD 是覆盖游戏画布的独立全屏组件，内部按可视区域百分比分配暂停、Score、Time 和挑战倒计时，不依赖主场景固定像素偏移。
- 左上暂停入口位于顶部斜线下方；Score 置于其下；Time 靠右；挑战倒计时仅在挑战模式显示并位于 Time 下方。
- HUD 仅在游戏对局可见，模式选择与结算卡显示期间隐藏。

## 保存与继续

- 暂停面板的最下方“保存并退出”从禁用改为可用。点击后即时关闭暂停层和局内画面，返回模式选择。
- 模式选择页只保留“经典模式”和“挑战模式”两个主按钮，不显示独立的继续按钮。点击某个模式且该模式存在可恢复存档时，在当前页面上方显示深色遮罩的“进度”确认弹窗。
- 弹窗使用紫色票券式标题栏、深蓝内容区、右上角关闭按钮和红/绿圆形操作按钮；提示文案为“上次玩到{score}，是否继续？”。`score` 直接显示上次存档的整数分数，不补零、不做多位数格式化。
- 点击红色叉号“不继续”时仅清除当前模式存档并立即开始该模式新局；点击绿色对号“继续”时恢复该模式的 `READY` 状态：经典从上一稳定回合开始，挑战把全部球放回顶部槽并使用保存时的剩余倒计时。关闭弹窗不改变任何存档或局内状态。
- 存档读取失败时直接清除对应损坏存档并开始所选模式新局，不影响另一模式或最高分记录。
