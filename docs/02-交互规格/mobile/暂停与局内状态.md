# 暂停与局内状态交互规格

> 所属功能域：局内流程、暂停与 HUD  
> 版本归属：V0.4 | 生命周期状态：已审核

## 入口与面板

- 局内 HUD 左上角显示深蓝圆形底、白色双竖线的暂停按钮；模式选择页和游戏结束后隐藏。
- 点击后，以半透明深色遮罩压暗整个对局；场地画面保留在遮罩后方，便于玩家确认当前进度。
- 弹层居中使用完整深蓝色圆角底板与青绿色圆角标题条；标题左侧为绘制的白色双竖线图标、右侧为“暂停”，不使用文字“Ⅱ”或镂空凹洞。右上角“×”和首枚青绿色“▶ 继续游戏”按钮均恢复游戏，第二枚“↻ 重新开始”按钮重置为相同模式的新局。
- 最底部显示“⌂ 保存并退出”按钮，使用较暗的青绿色与低对比文字；当前禁用且不响应点击，待续局存档完成后再启用。
- 弹层吞掉其余点击，玩家不能透过菜单继续拖拽瞄准或触发场内对象。

## 状态体验

| 操作 | 游戏世界 | 菜单 | 结果 |
|---|---|---|---|
| 点击暂停 | 立刻冻结 | 显示 | 保留当前球、棋盘与 HUD 数值 |
| 点击继续或关闭 | 恢复暂停前状态 | 隐藏 | 物理、正计时与挑战倒计时继续推进 |
| 点击重新开始 | 清除当前局 | 隐藏 | 以当前经典/挑战模式重置为新局 |
| 结束/返回模式选择 | 不可暂停 | 隐藏 | 不显示暂停入口 |

## 适配规则

- 面板宽度为可视区域约 `82%`，随竖屏和桌面调试窗口居中；标题条和按钮使用高对比白字，触控高度不小于 `48dp` 的视觉等效尺寸。
- 遮罩和菜单自身必须使用不受场景暂停影响的处理模式，确保继续按钮始终可点。
