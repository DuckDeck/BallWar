# Ball War 项目导览

> 面向后续开发者和 AI 的第一阶段代码导览。先从本页开始，按任务加载下方文档。  
> 版本归属：V0.1-MVP | 生命周期状态：已审核

Ball War 是一个 Godot 4.7 + GDScript 的离线竖屏弹球原型：球从顶部中央缺口向下发射，受重力和边界反弹影响，命中数字障碍；触底后沿可见路线回到同一发球点。

## 术语速查

- **活动球**：处于 `Ball.ACTIVE`、受重力和碰撞影响的球。
- **托底曲线**：场地最下方浅碗形曲线；不是物理碰撞体，而是回收触发线。
- **回收子状态**：`ROLLING`、`LIFTING`、`ROOF_RETURN`，用于回到顶部缺口。
- **发球缺口**：两条顶部斜线之间的开口；待发球、真实出生点和回收终点共享该位置。
- **定义 Resource**：用于描述某类球或障碍的静态数据；不应被本局运行状态污染。
- **命中协议**：包含来源、伤害、命中点和能力标签的统一受击约定，供未来特殊球与障碍复用。
