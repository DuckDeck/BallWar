# Ball War 项目概览

> 引擎：Godot 4.7  
> 语言：GDScript  
> 项目形态：离线 2D 客户端

## 入口与运行

- 主场景：`res://scenes/main.tscn`
- 配置入口：`res://project.godot`
- 运行要求：使用 Godot **4.7** 导入项目后运行主场景。
- 设计基准：1080×1920 竖屏，窗口以 Canvas Items 方式等比伸缩。

## 目录职责

| 目录 | 职责 |
|---|---|
| `scenes/` | 可组合的主场景、球和障碍场景。 |
| `scripts/` | 场景行为、物理、输入、程序绘制和局内流程。 |
| `resources/` | `.tres` 可调参数资源。 |
| `assets/` | 为后续图标、音效或已确认外部资源预留；T1 不放置位图资源。 |
| `docs/` | 需求、交互、技术规约、测试和项目说明。 |

## T1 最小能力

- 从顶部发射圆圈下方的任意屏幕位置以鼠标或单指点击、向下拖拽发射单球；球始终从顶部发射位出发。
- 球体持续受重力影响，在左右与顶部边界反弹；向上反弹后会减速回落，命中数字障碍物时扣除一点生命。
- 障碍物生命归零时增加 10 分；球回收或超时后恢复可发射状态。
- 场地、球、障碍、瞄准标记和分数文本均为程序绘制或 Godot 标准 UI，不依赖图片。

## 后续边界

多球、生成、奖励物、完整 HUD、暂停、动效、美术细化以及平台导出分别由 T2–T6 处理；不要在 T1 文件中提前加入这些规则。
